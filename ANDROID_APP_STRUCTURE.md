# Android Додаток - Структура проекту

## Технології
- **Kotlin** - мова програмування
- **Jetpack Compose** - UI фреймворк
- **Retrofit** - HTTP клієнт для REST API
- **Room Database** - локальна база даних для офлайн режиму
- **Hilt** - Dependency Injection
- **Coroutines & Flow** - асинхронність
- **Material Design 3** - дизайн система

## Структура проекту

```
app/
├── src/main/java/com/servicecenter/
│   ├── data/
│   │   ├── api/
│   │   │   ├── ApiService.kt          # Retrofit інтерфейс
│   │   │   └── ApiClient.kt           # Налаштування Retrofit
│   │   ├── local/
│   │   │   ├── database/
│   │   │   │   ├── AppDatabase.kt     # Room база даних
│   │   │   │   ├── entities/          # Entity класи
│   │   │   │   └── dao/                # DAO інтерфейси
│   │   │   └── repository/            # Репозиторії
│   │   └── models/                    # Data models
│   ├── domain/
│   │   └── usecases/                  # Use cases
│   ├── ui/
│   │   ├── theme/                      # Теми (Material Design 3)
│   │   ├── screens/
│   │   │   ├── home/                   # Головний екран
│   │   │   ├── repairs/                # Ремонти
│   │   │   ├── warehouse/              # Склад
│   │   │   ├── transactions/           # Транзакції
│   │   │   └── statistics/            # Статистика
│   │   ├── components/                 # Переіспользувані компоненти
│   │   └── navigation/                 # Навігація
│   ├── di/                             # Hilt модулі
│   └── MainActivity.kt
├── build.gradle.kts
└── AndroidManifest.xml
```

## Основні екрани

1. **Головний екран (Home)**
   - Пошук ремонтів
   - Швидкий доступ до функцій
   - Статус підключення до сервера

2. **Ремонти (Repairs)**
   - Список ремонтів з фільтрами
   - Створення нового ремонту
   - Редагування ремонту
   - Додавання товарів зі складу

3. **Склад (Warehouse)**
   - Перегляд товарів на складі
   - Пошук товарів

4. **Транзакції (Transactions)**
   - Список транзакцій
   - Фільтри по датах/категоріях

5. **Статистика (Statistics)**
   - Фінансова статистика
   - Графіки доходів/витрат

## Налаштування підключення

- Екран налаштувань для введення IP адреси сервера
- Збереження в SharedPreferences
- Автоматичне підключення при запуску

## Синхронізація

- Локальна база даних (Room) для офлайн роботи
- Синхронізація з сервером при підключенні
- Конфлікт-резолюція при одночасних змінах
