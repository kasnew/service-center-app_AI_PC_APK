# Service Center Android App

Android додаток для роботи з системою управління сервісним центром.

## Вимоги

- Android Studio Hedgehog | 2023.1.1 або новіша
- Android SDK 24+ (Android 7.0+)
- Kotlin 1.9.0+

## Встановлення

1. Відкрийте проект в Android Studio
2. Синхронізуйте Gradle файли
3. Запустіть на емуляторі або реальному пристрої

## Налаштування

1. Відкрийте налаштування додатку
2. Введіть IP адресу та порт сервера (наприклад: `http://192.168.1.100:3000`)
3. Натисніть "Підключити"

## Функціонал

- ✅ Пошук та прийом техніки
- ✅ Додавання товарів в квитанції зі складу
- ✅ Перегляд транзакцій
- ✅ Фінансова статистика
- ✅ Офлайн режим з синхронізацією
