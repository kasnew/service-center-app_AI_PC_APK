# Інструкція по встановленню Android додатку на телефон

## Варіант 1: Через ADB (Android Debug Bridge) - Рекомендовано

### Крок 1: Підготовка телефону

1. **Увімкніть "Режим розробника":**
   - Відкрийте **Налаштування** → **Про телефон**
   - Знайдіть **"Номер збірки"** або **"Версія Android"**
   - Натисніть на нього **7 разів** підряд
   - З'явиться повідомлення "Ви тепер розробник!"

2. **Увімкніть USB налагодження:**
   - Відкрийте **Налаштування** → **Для розробників**
   - Увімкніть **"USB налагодження"**
   - Дозвольте **"Налагодження USB"** (якщо з'явиться запит)

3. **Дозвольте встановлення з невідомих джерел:**
   - **Налаштування** → **Безпека** → **Невідомі джерела** (або **Налаштування** → **Для розробників** → **Встановлення через USB**)

### Крок 2: Підключення до комп'ютера

1. Підключіть телефон до комп'ютера через USB кабель
2. На телефоні дозвольте налагодження USB (з'явиться запит)

### Крок 3: Встановлення

**Автоматично (через скрипт):**
```bash
cd android-app
./install-to-phone.sh
```

**Вручну:**
```bash
cd android-app
adb devices  # Перевірка підключення
adb install -r app/build/outputs/apk/debug/app-debug.apk
```

## Варіант 2: Через файловий менеджер (Без USB)

### Крок 1: Отримайте APK файл

APK файл знаходиться тут:
```
android-app/app/build/outputs/apk/debug/app-debug.apk
```

### Крок 2: Передайте файл на телефон

**Варіанти передачі:**
- Через USB (як звичайний файл)
- Через Bluetooth
- Через Google Drive / Dropbox
- Через email
- Через месенджер (Telegram, WhatsApp)

### Крок 3: Встановлення на телефоні

1. Відкрийте файловий менеджер на телефоні
2. Знайдіть завантажений APK файл
3. Натисніть на нього
4. Якщо з'явиться попередження про невідомі джерела:
   - Натисніть **"Налаштування"**
   - Увімкніть **"Дозволити з цього джерела"**
   - Поверніться та натисніть **"Встановити"**
5. Після встановлення натисніть **"Відкрити"**

---

## Варіант 3: Через Android Studio

1. Відкрийте проект в Android Studio
2. Підключіть телефон через USB
3. Увімкніть USB налагодження на телефоні
4. В Android Studio натисніть зелену кнопку **"Run"** (▶) або `Shift + F10`
5. Виберіть ваш телефон зі списку пристроїв
6. Додаток автоматично встановиться та запуститься

---

## Перевірка встановлення

Після встановлення додаток буде доступний в меню додатків під назвою **"Service Center"**.

---

## Вирішення проблем

### "Пристрій не знайдено" (ADB)

1. Перевірте, чи увімкнено USB налагодження
2. Спробуйте інший USB кабель
3. Перевірте, чи встановлені драйвери USB для вашого телефону
4. Спробуйте інший USB порт

### "Встановлення заблоковано"

1. Перевірте налаштування безпеки:
   - **Налаштування** → **Безпека** → **Невідомі джерела**
   - Або **Налаштування** → **Для розробників** → **Встановлення через USB**

### "Помилка встановлення"

1. Переконайтеся, що на телефоні достатньо місця
2. Спробуйте видалити стару версію додатку (якщо вона є)
3. Перезавантажте телефон

## Розмір APK файлу

Поточний розмір: ~16 MB

---

## Оновлення додатку

Для оновлення просто встановіть нову версію APK поверх старої. Всі дані збережуться.
