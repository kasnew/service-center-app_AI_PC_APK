# Інструкція з налаштування Android додатку

## Вимоги

- Android Studio Hedgehog (2023.1.1) або новіша версія
- Android SDK 24+ (Android 7.0+)
- Kotlin 1.9.20+
- JDK 17+

## Встановлення

1. Відкрийте Android Studio
2. File → Open → виберіть папку `android-app`
3. Дозвольте Gradle синхронізувати залежності
4. Зачекайте, поки Gradle завантажить всі залежності

## Налаштування

### 1. Налаштування підключення до сервера

1. Запустіть Electron додаток
2. Відкрийте Settings → Синхронізація
3. Натисніть "Запустити сервер"
4. Скопіюйте IP адресу (наприклад: `http://192.168.1.100:3000`)

### 2. Налаштування в Android додатку

1. Запустіть Android додаток
2. Відкрийте "Налаштування"
3. Введіть URL сервера
4. Натисніть "Зберегти" та "Перевірити"

## Структура проекту

```
android-app/
├── app/
│   └── src/main/java/com/servicecenter/
│       ├── data/              # Data layer
│       │   ├── api/           # Retrofit API
│       │   ├── local/         # Room Database
│       │   └── models/        # Data models
│       ├── ui/                # UI layer
│       │   ├── screens/       # Екрани
│       │   ├── navigation/    # Навігація
│       │   └── theme/         # Теми
│       └── di/                # Dependency Injection
└── build.gradle.kts
```

## Важливі зауваження

1. **Офлайн режим**: Додаток працює офлайн, дані зберігаються локально
2. **Синхронізація**: При підключенні до сервера дані автоматично синхронізуються
3. **Конфлікти**: При одночасних змінах пріоритет мають дані з сервера
